### Описание проекта    
Анализ вакансий в базе данных hh.ru с целью создания рекомендательной системы для соискателей


**Что практикуем**     
Учимся писать оптимальные запросы на языке SQL.


### Краткая информация о данных
Для анализа нам предоставлено 5 взаимосвязанных таблиц, в которых хранятся данные по вакансиям и работодателям. Всего представлено 49197 вакансий от 23501 работодателя. Количество регионов (городов), по которым у нас есть сведения - 1362. Сфер деятельности - 294.

### Выводы:  
В ходе анализа выявлено несколько узких мест, для закрытия которых было сформулировано несколько рекомендаций. 1. Указание сферы деятельности компании сделать обязательным полем для работодателей. 2. Сделать обязательным заполнение зарплатной вилки: нижней, верхней, либо обеих границ (по усмотрению работодателя). Эти сведения позволят повысить информированность соискателей, облегчит выбор подходящей вакансии, а также позволит создать рекомендательную систему для соискателей.
